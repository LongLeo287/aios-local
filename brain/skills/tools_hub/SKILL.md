## Decision Flow

```
Nhận request
  → Identify category (Security / Session / Skill / Project / MCP)
  → Route to correct tool
  → Confirm params với user nếu cần
  → Execute, report result
  → Log nếu có concern bảo mật
```

## Pre-flight Checks

Trước khi chạy bất kỳ tool nào:
1. **Security tools**: Chỉ chạy trên repos đã clone vào QUARANTINE/
2. **Backup tools**: Confirm path tồn tại
3. **MCP servers**: Check port không conflict
4. **Skill tools**: Verify skill name tồn tại trong registry

# Constraints

- MCP server cần Node.js — check `node --version` trước khi run
- Full tool list luôn up-to-date tại: `registry/TOOLS_REGISTRY.md`
