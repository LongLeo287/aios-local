## ID: 2026-03-26-IDEA-01 
- **Tên Dự Án:** AI OS Eye (Cặp Mắt Viễn Vọng Chrome Extension)
- **Domain Đề Xuất:** `system/integrations/chrome_extension_eye/`
- **Mục tiêu:** Tiện ích mổ bụng Dữ Liệu Chrome. Gắn liền vào AI OS để bất cứ thứ gì Sếp bôi đen trên trình duyệt đều được bắn thẳng vào kho `storage/vault` làm RAG Context.
- **Vũ Khí Code:** Cần dùng Google Developer Knowledge MCP cào Docs API Chrome Extension (Manifest V3) để sinh Code tự động.

## ID: 2026-03-26-IDEA-04
- **Tên Dự Án:** Android Companion App (Bót Điều Khiển Di Động)
- **Domain Đề Xuất:** `system/ui/android_companion/`
- **Mục tiêu:** Một ứng dụng Native siêu nhẹ viết bằng Jetpack Compose cho điện thoại Android. Thiết bị Android kết nối vào mạng Local Wifi, chọc thẳng vào AI OS để cho phép Sếp điều khiển Agent, xem Log Console ngay trên giường tấu sớ.
- **Vũ Khí Code:** Cần dùng Google Developer Knowledge MCP tra cứu Android Docs cho Jetpack Compose UI và OkHttp / Retrofit Networking.
